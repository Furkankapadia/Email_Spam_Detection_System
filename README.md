# Spam Email Detector  
A Machine Learning project that detects whether a message is **Spam** or **Ham (Not Spam)** using NLP and the Multinomial Naive Bayes algorithm. 

## Technologies
- Python
- Scikit-learn
- NLTK 
- Pandas
- HTML/CSS/JavaScript

- ## Accuracy : 97.31%
